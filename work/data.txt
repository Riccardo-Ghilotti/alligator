<strong>A.L.L.I.G.A.T.O.R. - Automated Learning and Linking for Intelligent Graph-based Association of Tabular Objects and Relationships</strong> is a project that performs entity linking over tabular data with the support of a ML algorithm. 
Through an API specification, the tool allows for the performance of entity linking against <a href='https://www.wikidata.org/'>Wikidata</a>.<br>
