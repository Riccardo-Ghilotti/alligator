<strong>A.L.L.I.G.A.T.O.R. - Automated Learning and Linking for Intelligent Graph-based Association of Tabular Objects and Relationships</strong> is a project that performs entity linking over tabular data with support of a ML algorithm.
Through an API specification, the tool allows to perform entity linking against  <a href="https://www.wikidata.org/">Wikidata</a>
